Web Units Guide

be sure to resize the browser and see how certain elements change shapes. All of the sizes presented can be used for other properties such as paddings, margins etc... But be careful or you will end up with some unexpected results.

%
% units take a percentage value off their parent element
Inch
inches are fixed units. 1inch = 2.54cm
cm
cm are also fixed units, 1cm = 10mm
mm
you guessed it, milimeters are fixed as well.
em
em units are really interesting. They are related to the font size of the parent element. My Box div has a font size property of 30px, so 1em = 30px.
rem
rem units are similar to em units, except they are relative to the document root font size.
ex
1 ex is the x-height of a font (x-height is usually about half the font-size)
pt
1pt(point) = 1/72 inch
pc
1pc/pica = 12pt
px
We all know pixels right?
Now this is really cool!

vw and vh units are viewport based and will actually scale when you rezise the window. See it for yourself :)!

vw
1vw = 1% width of the viewport. This will actually scale when we resize the window. Cool, huh?
vh
1vh = The same as vw, but it takes 1% of the height of the viewport.
